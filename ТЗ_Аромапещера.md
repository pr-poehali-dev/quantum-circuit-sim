# ТЕХНИЧЕСКОЕ ЗАДАНИЕ
## Веб-приложение «Аромат и Ум» (Аромапещера)

**Версия:** 1.0  
**Дата:** 17 мая 2026

---

## 1. ОБЩЕЕ ОПИСАНИЕ

Веб-приложение с атмосферным интерфейсом для управления аудиозаписями и компиляциями. Три типа пользователей: Администратор, Свои Клиенты (создают записи), Чужие Клиенты (слушают готовые материалы).

**Стек:** React + TypeScript, Python (бэкенд), PostgreSQL

---

## 2. СТРАНИЦЫ И ФУНКЦИОНАЛ

### 2.1 Главная страница (`/`)
- Полноэкранный горный пейзаж с анимацией (облака, цветы)
- **5 кликабельных зон:**
  - Гора Админа → `/admin`
  - Аромапещера → `/topics`
  - Свои Клиенты → `/login`
  - Чужие Клиенты → `/listen`
  - Домик → `/house` (регистрация)
- Секция «Как это работает» (3 шага)
- Секция «Панель администратора»
- Шапка с навигацией и подвал

---

### 2.2 Страница выбора вида (`/topics`) — Экран камня
- Полноэкранный фон пещеры
- **Камень с выбором:**
  - 🕯️ **Сказочный вид** — переход к сцене пещеры
  - 📋 **Классический вид** — переход на `/topics-classic`
- Кнопка «← На главную»

---

### 2.3 Аромапещера — Сказочный вид (`/topics` → шаг enter/login/inside)

**Этап 1 — Вход в пещеру:**
- Анимированная сцена входа
- Кнопка «Войти в пещеру»

**Этап 2 — Авторизация:**
- Форма: Логин + Пароль
- Отправка на бэкенд, обработка ошибок
- Интерактивный камень с правилами (диалог)

**Этап 3 — Список тем:**
- Темы с количеством записей и статусом компиляции
- Анимация факела (реагирует на скролл)
- Фоновая атмосферная музыка

---

### 2.4 Аромапещера — Классический вид (`/topics-classic`)

**Форма входа:**
- Поля: Логин, Пароль
- Кнопка «Войти» с индикатором загрузки
- Отображение ошибок

**После входа — список тем:**
- Карточки с названием, количеством записей, статусом
- Приветствие: «Привет, {имя}»
- Кнопка «← На главную»

---

### 2.5 Панель Своих Клиентов (`/login`)
- Кнопка «🎤 Записать новый текст»
- Список своих записей (Play, Delete, название, дата, длительность)
- Список компиляций с возможностью прослушивания

---

### 2.6 Чат Своих Клиентов (`/chat`)

**Этап 1 — Ввод имени:**
- Поле «Как вас зовут?», сохранение в localStorage

**Этап 2 — Чат:**
- Обновление сообщений каждые 5 секунд
- Отправка по Enter / кнопке
- Автоскролл к последнему сообщению
- Время у каждого сообщения

---

### 2.7 Панель Чужих Клиентов (`/listen`)
- Список готовых компиляций (без регистрации)
- Play/Pause с анимированным визуализатором
- Название, длительность, количество прослушиваний

---

### 2.8 Регистрация (`/house` → `/register`)

**Изба (/house):**
- Интерактивная сцена, кликабельный стол → переход к форме

**Форма (/register):**
- Поля: Имя, Телефон, Email
- После отправки — экран подтверждения

---

### 2.9 Панель администратора (`/admin`)
- Управление темами (создать, редактировать, удалить)
- Управление клиентами (пригласить, заблокировать)
- Модерация записей
- Запуск ИИ-компиляции
- Выгрузка аудио

---

## 3. БЭКЕНД (Cloud Functions)

| Функция | Метод | Назначение |
|---|---|---|
| `cave-login` | POST | Авторизация клиентов (MD5-хэш пароля, PostgreSQL) |
| `chat` | GET / POST | Получение и отправка сообщений чата |

---

## 4. БАЗА ДАННЫХ (PostgreSQL)

| Таблица | Назначение |
|---|---|
| `cave_clients` | Логины и пароли клиентов Аромапещеры |
| `chat_messages` | Сообщения чата (автор, текст, время) |

---

## 5. ДИЗАЙН

- **Общая стилистика:** Атмосферная, природная (пещеры, горы, изба)
- **Цвета:** Янтарный, тёмно-коричневый, чёрный — для сказочного вида; светло-серый — для классического
- **Шрифты:** Georgia (декоративный), системный sans-serif (интерфейс)
- **Анимации:** Framer Motion (переходы, появления, hover-эффекты)
- **Иконки:** Lucide React + Emoji
